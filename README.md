## 1.배경  
 - 안녕하세요 여러분!🙌 집값 예측 경진대회에 오신 것을 환영합니다.
train데이터 및 test데이터 총합 2,700개의 집값 데이터가 주어집니다.

 - 다른 사람들과 실력을 겨뤄보며 데이터 분석 대회의 즐거움을 느껴보세요.
